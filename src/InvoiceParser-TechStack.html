InvoiceParser — Technology Stack

Project: InvoiceParser  |  Generated: April 10, 2026

Runtime & Language

  • C# / .NET 7.0 — all projects target net7.0
  • Visual Studio 2022 (v17) — IDE / solution format

Architecture

  • Clean / Layered Architecture with separated concerns:
    • InvoiceParser.Core — Domain logic & services
    • InvoiceParser.Infrastructure — Data access & repositories
    • InvoiceParser.Web — Presentation layer (MVC)
    • TestVerizon — Console test harness

Web Framework

  • ASP.NET Core MVC — Controllers + Razor Views (.cshtml), server-rendered

Frontend / UI

  • Bootstrap 5 — Responsive CSS framework
  • jQuery — DOM manipulation & AJAX
  • Select2 (v4.1.0-rc.0) — Enhanced dropdown controls with Bootstrap 5 theme

Database

  • Microsoft SQL Server — Two databases: datamart (primary) and iPath (secondary)
  • Entity Framework Core 7 — ORM for data access
    • Microsoft.EntityFrameworkCore.SqlServer
    • Microsoft.EntityFrameworkCore.Design

AI / Machine Learning

  • OpenAI-compatible API — via custom provider (devs.ai / AppDirect) for AI-powered invoice parsing
  • ML.NET (v3.0.1) — On-device machine learning for invoice field prediction

PDF Processing

  • PdfPig (UglyToad.PdfPig v1.7.0-custom-5) — PDF text extraction (custom/forked build)

Key Application Services

Service Purpose
PdfTextExtractorServiceExtracts raw text from uploaded PDF invoices
AiInvoiceParserSends invoice text to AI API for structured parsing
GenericInvoiceParserRule-based parsing for generic invoices
VerizonWirelessParserSpecialized parser for Verizon Wireless invoices
InvoiceMLServiceML.NET-based field prediction model
FeedbackAgentAI-driven feedback / correction loop
ChargeValidationServiceAI-powered charge validation
RuleLearningServiceLearns parsing rules from user corrections
FeedbackProcessorProcesses user feedback to improve parsing accuracy
Summary: An ASP.NET Core 7 MVC application that parses PDF invoices using a combination of rule-based parsing, ML.NET models, and LLM/AI APIs. Results are stored in SQL Server via Entity Framework Core and presented through a server-rendered UI built with Bootstrap 5, jQuery, and Select2.
